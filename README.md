# Fundamentos del lenguaje GO

## Instalación
Para instalar GO debemos seguir los siguientes pasos:

1. Instalarlo desde el siguiente enlace: https://go.dev/dl (en mi caso Windows)
- Verificamos a traves de la consola de comandos si está instalado con el comando <b>go version</b> (debe salir un número de versión, o mensaje parecido a este: go version go1.26.3 windows/amd64)
2. Vamos a Visual Studio Code e instalamos la extensión de GO
