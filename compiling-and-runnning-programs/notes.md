# ¿Como compilar y correr un programa con GO?

## Pasos
1. Abrir la terminal en la carpeta donde está ubicado el archivo y escribir la siguiente instrucción:
* go run nombre-archivo.go

2. El programa mostrará el mensaje : "Hello, World!"

## Crear un binario usando GO
Ejecutando la siguiente instrucción: 
* go build hello.go
Nos termina creando un archivo ejecutable (.exe) y podemos ejecutarlo
